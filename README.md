# Personal Website

This is a static personal portfolio website that you can upload to GitHub and host with GitHub Pages, Vercel, or Netlify.

## Files

- `index.html` - Main website page
- `styles.css` - Website styling and responsive layout
- `script.js` - Mobile menu and current year behavior
- `assets/sandeep-singh-resume.txt` - Sample resume placeholder

## Customize Before Publishing

Update these sample values:

- Name and headline in `index.html`
- Email address in the contact section
- LinkedIn and GitHub links
- Project names, descriptions, and demo links
- Blog post titles and dates
- Replace `assets/sandeep-singh-resume.txt` with your real resume, preferably a PDF

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload the contents of this `personal-website` folder.
3. Go to repository `Settings` > `Pages`.
4. Choose the main branch and root folder.
5. Add your custom domain in the Pages settings.
6. In your domain provider DNS, add the records GitHub shows you.

## Publish With Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Use the default static site settings.
4. Add your `.com` domain in Vercel project settings.
5. Update DNS records at your domain provider as Vercel instructs.
